## Iniciar projeto

Clone e projeto e entre na pasta.

```bash
$ git clone https://github.com/Luis-Felipe-N/dashgo..git
$ cd dashgo.
```

Siga os passos abaixo: 
```bash
# Instale as dependencias
$ yarn
$ npm install

# Iniciar projeto
$ yarn start
$ npm start
```
